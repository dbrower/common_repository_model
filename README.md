# CommonRepositoryModel

Notre Dame's Common Fedora Repository Model

## Installation

Add this line to your application's Gemfile:

    gem 'common_repository_model'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install common_repository_model
